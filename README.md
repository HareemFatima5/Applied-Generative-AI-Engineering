# Week 3 – NLP & Text Classification Fundamentals

This repository contains my Week 3 learning work for the Applied Generative AI Engineering Internship. 
The focus of this week is understanding the fundamentals of Machine Learning workflows, Natural Language Processing (NLP), text classification, text vectorization, machine learning models for text
and evaluation metrics.

## Overview

During this week, I researched the core concepts behind text classification and traditional NLP pipelines. The work includes short articles explaining each topic along with Python code examples wherever applicable.
